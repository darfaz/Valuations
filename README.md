# Dar Fazulyanov - Valuation Expert

Personal website showcasing valuation expertise and founder experience.

## About

A modern, animated single-page website built with:
- Pure HTML5, CSS3, JavaScript (no frameworks)
- Smooth scroll animations
- Responsive design
- Dark theme with accent colors

## Setup

1. Add your profile photo to `assets/dar-photo.jpg`
2. Update contact email in `index.html`
3. Deploy to any static host (GitHub Pages, Netlify, Vercel)

## Deployment

### GitHub Pages
```bash
# Enable in repo Settings > Pages > Source: main branch
```

### Netlify
Just drag the folder to netlify.com/drop

### Custom Domain
Add a `CNAME` file with your domain.

## Customization

Colors are defined as CSS variables in `styles.css`:
```css
:root {
    --color-accent: #4f46e5;      /* Primary accent */
    --color-highlight: #22c55e;    /* Green highlight */
}
```
